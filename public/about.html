About Us

Welcome to Adopt-a-Pet, a project created to demonstrate a full-stack web application.

Our mission is to connect loving homes with animals in need. We believe every pet deserves a second chance, and we've built this platform to make that connection easier.

This project is built with:

  • Frontend: HTML, CSS, and Vanilla JavaScript
  • Backend: Node.js and Express
  • Database: MongoDB Atlas
  • Authentication: JWT (JSON Web Tokens) & bcrypt
  • Deployment: Vercel
